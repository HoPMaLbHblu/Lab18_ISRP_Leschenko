## Описание (что изучили)

В ходе лабораторной работы были изучены основы языка JavaScript:
- место JS в веб-разработке (frontend, backend с Node.js);
- отличие от C#: динамическая типизация, интерпретируемость, отсутствие точки входа;
- подключение JS к HTML-странице (тег `<script>`);
- объявление переменных с помощью `let`, `const`, `var`;
- основные типы данных: string, number, boolean, undefined, null, object, symbol;
- оператор `typeof` и его особенности (например, `typeof null === "object"`);
- явное и неявное преобразование типов;
- строгое (`===`) и нестрогое (`==`) сравнение;
- работа с браузерной консолью (DevTools);
- запуск JS-кода вне браузера с помощью Node.js;
- условные операторы (`if/else`, `switch`, тернарный оператор);
- сравнение синтаксиса и поведения JavaScript с C#.

## Итоговая таблица: JavaScript vs C#

| Аспект               | C#                                          | JavaScript                                   |
|----------------------|---------------------------------------------|----------------------------------------------|
| **Типизация**        | Статическая                                 | Динамическая                                 |
| **Компиляция**       | Да (в IL-код)                               | Нет (интерпретация / JIT)                    |
| **Точка входа**      | `Main()`                                     | Отсутствует                                  |
| **Переменные**       | `int x = 5;`                                 | `let x = 5;`                                 |
| **Константы**        | `const int X = 5;`                           | `const X = 5;`                               |
| **Вывод в консоль**  | `Console.WriteLine()`                        | `console.log()`                              |
| **Проверка типа**    | Во время компиляции                          | `typeof`                                     |
| **Сравнение**        | `==` (по значению)                           | `===` (строгое)                              |
| **Условия**          | `if/else/switch`                             | `if/else/switch` + тернарный                 |
| **Циклы**            | `for/while/do-while/foreach`                 | `for/while/do-while/for...of`                |
| **Функции**          | Методы с типами                              | Функции без типов                            |
| **Массивы**          | `int[] arr = new int[5];`                    | `let arr = [];`                              |
| **Объекты**          | Требуют класс                                | Можно создать литералом                      |
| **ООП**              | Классическое                                 | Прототипное (классы – синтаксический сахар) |
| **Null**             | `null`                                       | `null` и `undefined`                         |
| **Строгость**        | Высокая (компилятор)                         | Низкая (нужна дисциплина)                    |
